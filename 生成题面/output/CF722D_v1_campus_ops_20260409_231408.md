# 社团技能传承计划

> Variant: `CampusOps_CountingGenerators` / `v1` / theme `campus_ops`

## 题目描述

校园社团正在筹备新一届招新。现有 n 名正式成员，他们的最终技能等级分别为 y1, y2, ..., yn（互不相同）。社团规定，每位新成员必须从一名候选人开始培养。候选人的初始技能等级为正整数，且所有候选人的初始等级互不相同。

培养规则如下：若某候选人当前技能等级为 v，经过一次基础训练可提升至 2v，经过一次进阶训练可提升至 2v+1。训练可进行任意多次。若一名候选人经过若干次训练后恰好达到某正式成员的技能等级，则称该候选人成功传承给了该成员。

已知 n 名正式成员的技能等级集合 Y = {y1, y2, ..., yn}。我们需要招募恰好 n 名候选人，构成初始集合 X，使得 X 中的每个元素都能通过上述训练规则一对一地传承给 Y 中的某个元素。

为了控制招新门槛，我们希望初始集合 X 中的最高技能等级尽可能低。记所有合法初始集合 X 中最大元素的最小可能值为 M*。现在，请你计算有多少种不同的初始集合 X，满足 X 能一对一传承给 Y，且 X 中的最大元素恰好等于 M*。由于答案可能很大，请输出其对给定模数 P 取模的结果。

两个集合被视为不同，当且仅当它们包含的元素集合不同。

## 输入格式

第一行包含两个正整数 n 和 P，分别表示正式成员人数和模数。
第二行包含 n 个互不相同的正整数 y1, y2, ..., yn，表示正式成员的技能等级。

## 输出格式

输出一个整数，表示满足条件的初始集合 X 的数量对 P 取模的结果。

## 数据范围与限制

- 1 <= n <= 50000
- 1 <= yi <= 10^9，且所有 yi 互不相同
- 1 <= P <= 10^9+7
- 时间限制：2.0 秒
- 空间限制：512 MB

## 样例

### 样例 1

**输入**

```text
2 1000000007
3 5
```

**输出**

```text
1
```

**说明**

正式成员等级为 {3, 5}。通过训练规则可知，3 的祖先为 {3, 1}，5 的祖先为 {5, 2, 1}。能生成 {3, 5} 且最大元素最小的初始集合为 {1, 2}，此时 M*=2。满足最大元素恰好为 2 的集合仅有 {1, 2} 一种，故输出 1。

### 样例 2

**输入**

```text
3 998244353
4 5 6
```

**输出**

```text
1
```

**说明**

正式成员等级为 {4, 5, 6}。经计算，理论最小最大初始等级 M*=3。唯一满足条件的初始集合为 {1, 2, 3}，故输出 1。

## 说明

1. 集合 X 生成集合 Y 的定义等价于：存在一个从 Y 到 X 的双射 f，使得对任意 y 属于 Y，x=f(y) 是 y 在完全二叉树（根为 1，节点 v 的父节点为 floor(v/2)）中的祖先。
2. M* 由输入集合 Y 唯一确定，可通过贪心策略预处理得到。本题要求统计所有满足 max(X)=M* 的不同集合 X 的数量，而非仅构造一组解。
3. 结果需对输入的 P 取模。若 P=1，则输出 0。
