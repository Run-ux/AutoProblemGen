# 校园门禁密钥分配

> Variant: `CampusKeyXORCount` / `v1` / theme `campus_ops`

## 题目描述

大学后勤部正在为新建的社团活动中心配置智能门禁系统。该中心由若干房间和连接它们的走廊构成，整体布局恰好形成一棵树结构。其中，仅与一条走廊相连的房间称为“活动室”（即树中的叶子节点），其余为“中转厅”。

每条走廊需要安装一把电子密钥，密钥类型仅有两种：标准型（记为 0）和通用型（记为 1）。出于安全审计与预算管控要求，系统需满足以下两条规则：
1. 任意两个不同的活动室之间，沿唯一连通路径经过的所有走廊密钥类型按位异或（XOR）的结果必须为 0。
2. 全校发放的通用型密钥（权值为 1 的边）总数，对给定的正整数 P 取模后，必须恰好等于给定的余数 K。

后勤部希望知道，在满足上述所有规则的前提下，共有多少种不同的密钥分配方案？由于方案数可能极大，请输出其对 10^9+7 取模后的结果。

注：两种方案被视为不同，当且仅当至少存在一条走廊分配的密钥类型不同。

## 输入格式

第一行包含三个正整数 N, P, K，分别表示房间总数（节点数）、模数 P 和目标余数 K。
接下来 N-1 行，每行包含两个正整数 u, v，表示房间 u 与房间 v 之间有一条走廊相连。保证输入构成一棵树。

## 输出格式

输出一个整数，表示满足条件的密钥分配方案总数对 10^9+7 取模的结果。

## 数据范围与限制

- 3 ≤ N ≤ 10^5
- 2 ≤ P ≤ 100
- 0 ≤ K < P
- 1 ≤ u, v ≤ N，保证输入构成一棵树
- 时间限制：2.0 秒
- 空间限制：512 MB

## 样例

### 样例 1

**输入**

```text
3 2 0
1 2
1 3
```

**输出**

```text
2
```

**说明**

树结构为 2-1-3。叶子节点为 2 和 3。路径异或为 0 要求边(1,2)与边(1,3)权值相同。合法方案有 (0,0) 和 (1,1)，对应权值为 1 的边数分别为 0 和 2。两者模 2 余数均为 0，满足 K=0，故共 2 种方案。

### 样例 2

**输入**

```text
5 3 0
1 2
2 3
2 4
1 5
```

**输出**

```text
3
```

**说明**

叶子节点为 3, 4, 5。约束要求边(2,3)与(2,4)权值相同，且边(2,3)^边(1,2)^边(1,5)=0。设 a=w(2,3)=w(2,4), b=w(1,2), c=w(1,5)，则 c=a^b。枚举 (a,b) 可得四种组合，对应权值 1 的总数分别为 0, 2, 3, 3。其中模 3 余 0 的有 3 种，故答案为 3。

## 说明

结果对 10^9+7 取模。题目中的“活动室”严格指树中度数为 1 的节点。计数基于边权向量空间，不进行对称性约简。
