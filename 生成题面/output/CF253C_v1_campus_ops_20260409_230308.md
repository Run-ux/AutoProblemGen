# 阶梯教室的通行方案

> Variant: `CampusNavCount` / `v1` / theme `campus_ops`

## 题目描述

校园新落成的阶梯教室共有 N 层，每层的座位排布并不完全对齐。为了方便管理，我们将第 r 层的座位从左至右编号为 1 到 L_r。一名新生需要从起始座位 (r1, c1) 前往目标座位 (r2, c2)。

教室内的移动遵循以下规则：
1. 左右移动：在同一层内，可以向左或向右移动一个座位。若移动后超出该层座位范围 [1, L_r]，则该操作无效。
2. 上下移动：可以向上或向下移动一层。由于各层座位数不同，当从第 r 层移动到第 r' 层时，若目标层的座位数 L_{r'} 小于当前列号 c，新生会自动调整至目标层的最后一个座位（即列号变为 min(c, L_{r'})）；否则保持原列号不变。

学校希望统计所有步数最少的合法移动指令序列的数量。由于方案可能非常多，请将结果对给定的模数 M 取余后输出。

## 输入格式

第一行包含一个整数 N，表示教室层数。
第二行包含 N 个整数 L_1, L_2, ..., L_N，表示每层的座位数。
第三行包含四个整数 r_1, c_1, r_2, c_2，分别表示起点与终点的层号与列号。
第四行包含一个整数 M，表示模数。

## 输出格式

输出一个整数，表示最短移动指令序列的数量对 M 取模的结果。

## 数据范围与限制

- 1 <= N <= 1000
- 1 <= L_i <= 1000
- 1 <= r_1, r_2 <= N, 1 <= c_1 <= L_{r_1}, 1 <= c_2 <= L_{r_2}
- M = 1000000007
- 时间限制：1.0 秒
- 空间限制：256 MB

## 样例

### 样例 1

**输入**

```text
3
2 3 2
1 1 3 2
1000000007
```

**输出**

```text
3
```

**说明**

最短步数为 3。三条最短指令序列分别为：下-下-右、下-右-下、右-下-下。

### 样例 2

**输入**

```text
3
3 1 3
1 2 3 2
1000000007
```

**输出**

```text
1
```

**说明**

由于第 2 层仅有 1 个座位，从第 1 层任意位置向下移动都会强制对齐到第 2 层第 1 座。最短路径唯一：下-下-右。

## 说明

指令序列不同当且仅当在某一步的操作方向不同。所有计数均在模 M 意义下进行线性叠加，需严格保证状态转移仅沿最短距离递增方向进行，避免重复计数或循环依赖。保证起点与终点坐标合法。
