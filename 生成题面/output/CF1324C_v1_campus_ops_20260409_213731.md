# 校园走廊定向通行

> Variant: `CampusCorridorNav` / `v1` / theme `campus_ops`

## 题目描述

校园主楼有一条长度为 n 的笔直走廊，地面标有 n 个位置，编号从 1 到 n。每个位置 i 的地面上都印有一个方向箭头 s[i]（L 表示向左，R 表示向右）。现在，你需要从走廊入口（位置 0）出发，前往走廊尽头的仓库（位置 n+1）。

由于你搬运着沉重的社团物资，你的步幅受到严格限制：每次跳跃的跨度不能超过 k 个单位长度。同时，你必须遵守地面的箭头指示：
- 在起点 0 时，你只能向右跳跃。
- 当你落在位置 i (1 <= i <= n) 时，若 s[i] 为 L，你下一次只能向左跳；若 s[i] 为 R，你下一次只能向右跳。

每次跳跃必须精确落在整数坐标上。请判断是否存在一条合法路径到达 n+1。如果存在，请输出你依次经过的所有位置坐标（包含起点 0 和终点 n+1）；如果不存在，请找出一个导致无法通行的阻塞区间 [l, r]，该区间需满足：s[l..r] 全部为 L，且区间长度 r-l+1 >= k。该区间将作为无法通行的直接证据。

## 输入格式

第一行包含一个整数 t，表示测试用例的数量。
每个测试用例的第一行包含两个整数 n 和 k。
第二行包含一个长度为 n 的字符串 s，仅由字符 L 和 R 组成。

## 输出格式

对于每个测试用例：
如果存在合法路径，输出一行，按顺序输出路径上的所有坐标，用空格分隔。
如果不存在合法路径，输出一行，包含两个整数 l 和 r，表示阻塞区间的左右端点。

## 数据范围与限制

- 1 <= t <= 10000
- 1 <= n <= 200000
- 1 <= k <= n + 1
- 字符串 s 仅包含字符 L 和 R
- 所有测试用例的 n 之和不超过 200000
- 时间限制：1.0 秒
- 空间限制：256 MB

## 样例

### 样例 1

**输入**

```text
1
5 2
RRLRL
```

**输出**

```text
0 2 4 6
```

**说明**

从 0 出发，第一步跳至 2（跨度 2，s[2]=R），第二步跳至 4（跨度 2，s[4]=R），第三步跳至 6（跨度 2，到达终点）。路径合法。

### 样例 2

**输入**

```text
1
4 2
RLLL
```

**输出**

```text
2 4
```

**说明**

从 0 只能跳至 1（s[1]=R）。从 1 出发最大跨度为 2，可到达 2 或 3，但 s[2] 和 s[3] 均为 L，下一步只能向左跳，无法继续向右推进。位置 2 到 4 构成连续 L 区间，长度为 3 >= k，故输出 2 4。

## 说明

若存在多个合法的阻塞区间，输出任意一个满足条件的区间即可。路径输出必须严格包含起点 0 与终点 n+1，且相邻坐标差的绝对值不超过 k。
