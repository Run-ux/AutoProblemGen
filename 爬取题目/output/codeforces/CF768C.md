Input
First line consists of three integers n, k, x (1 ≤ n ≤ 10 5, 0 ≤ k ≤ 10 5, 0 ≤ x ≤ 10 3) — number of rangers Jon has, the number of times Jon will carry out the operation and Jon's favourite number respectively. Second line consists of n integers representing the strengths of the rangers a 1, a 2, ..., a n (0 ≤ a i ≤ 10 3).

Output
Output two integers, the maximum and the minimum strength of the rangers after performing the operation k times.

Constraints
time limit per test 4 seconds
memory limit per test 256 megabytes
