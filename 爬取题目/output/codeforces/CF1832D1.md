Input
The first line contains two integers $n$ and $q$ ($1 \le n, q \le 1000$) — the number of elements in the array and the number of queries. The second line contains $n$ integers $a_1, a_2, \dots, a_n$ ($1 \le a_i \le 10^9$). The third line contains $q$ integers $k_1, k_2, \dots, k_q$ ($1 \le k_j \le 10^9$).

Output
For each query, print a single integer — the largest minimum that the array can have after you apply exactly $k$ operations to it.
