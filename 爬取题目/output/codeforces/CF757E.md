Bash got tired on his journey to become the greatest Pokemon master. So he decides to take a break and play with functions.

Input
The first line contains an integer q (1 ≤ q ≤ 10 6) — the number of values Bash wants to know. Each of the next q lines contain two integers r and n (0 ≤ r ≤ 10 6, 1 ≤ n ≤ 10 6), which denote Bash wants to know the value f r (n).

Output
Print q integers. For each pair of r and n given, print f r (n) modulo 10 9 + 7 on a separate line.

time limit per test 3 seconds
