D. Polyline

There are three points marked on the coordinate plane. The goal is to make a simple polyline, without self-intersections and self-touches, such that it passes through all these points. Also, the polyline must consist of only segments parallel to the coordinate axes. You are to find the minimum number of segments this polyline may consist of.

Input
Each of the three lines of the input contains two integers. The i -th line contains integers x i and y i (- 10 9 ≤ x i, y i ≤ 10 9) — the coordinates of the i -th point. It is guaranteed that all points are distinct.

Output
Print a single number — the minimum possible number of segments of the polyline.

Constraints
time limit per test 1 second
memory limit per test 256 megabytes
