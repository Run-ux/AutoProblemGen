D. Yaroslav and Divisors

Yaroslav has an array p = p 1, p 2, ..., p n (1 ≤ p i ≤ n), consisting of n distinct integers. Also, he has m queries:

Help Yaroslav, answer all his queries.

Input
The first line contains the integers n and m (1 ≤ n, m ≤ 2·10 5). The second line contains n distinct integers p 1, p 2, ..., p n (1 ≤ p i ≤ n). The following m lines contain Yaroslav's queries. The i -th line contains integers l i, r i (1 ≤ l i ≤ r i ≤ n).

Output
Print m integers — the answers to Yaroslav's queries in the order they appear in the input. Please, do not use the %lld specifier to read or write 64-bit integers in C++. It is preferred to use the cin, cout streams or the %I64d specifier.

Constraints
time limit per test 2 seconds
memory limit per test 256 megabytes
