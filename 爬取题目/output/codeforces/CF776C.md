C. Molly's Chemicals

Molly Hooper has n different kinds of chemicals arranged in a line. Each of the chemicals has an affection value, The i -th of them has affection value a i.

Molly wants Sherlock to fall in love with her. She intends to do this by mixing a contiguous segment of chemicals together to make a love potion with total affection value as a non-negative integer power of k. Total affection value of a continuous segment of chemicals is the sum of affection values of each chemical in that segment.

Help her to do so in finding the total number of such segments.

Input
The first line of input contains two integers, n and k, the number of chemicals and the number, such that the total affection value is a non-negative power of this number k. (1 ≤ n ≤ 10 5, 1 ≤ | k | ≤ 10). Next line contains n integers a 1, a 2, ..., a n (- 10 9 ≤ a i ≤ 10 9) — affection values of chemicals.

Output
Output a single integer — the number of valid segments.

Constraints
time limit per test 2.5 seconds
memory limit per test 512 megabytes
